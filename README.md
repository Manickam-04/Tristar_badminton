# 🏸 Badminton Slot Booking Application

A Flask-based web application for managing badminton court slot bookings.

## Features

- User Registration & Login
- Slot Booking Management
- Booking History
- Admin Dashboard
- SQLite Database

## Tech Stack

- Flask
- Python
- SQLite
- HTML/CSS/JavaScript

## Project Structure

```text
badminton-slot-booking/
│
├── static/
├── templates/
├── app.py
├── requirements.txt
├── bookings.db
└── README.md
```

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```
